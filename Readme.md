- JWT Authentication (Register + Login)
- Role-based Access (user & admin)
- Full CRUD for Tasks
- Password hashing with bcrypt
- MongoDB + Mongoose
- React Frontend with beautiful pure CSS
- Swagger Documentation with Swagger

## Tech Stack
**Backend**: Node.js, Express, MongoDB, JWT, bcrypt  
**Frontend**: React, Axios, Pure CSS (no Tailwind)
